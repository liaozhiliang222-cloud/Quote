# 研价 · 调研报价助手

面向市场调研公司的本地优先 PWA 报价工具。项目、价格库、成本与利润数据默认保存在浏览器 IndexedDB，不会自动上传。

## 当前能力

- 线上定量、线下定量、深度访谈、焦点小组/座谈会、专家访谈、桌面研究、综合研究 7 类报价模板
- 单项成本、建议售价、价格区间与各职级研究人员人天配置
- 项目成本覆盖、自定义成本、利润情景与最低安全价
- 报价状态、报价版本及历史价格快照
- 客户版/内部版 XLSX 与打印/PDF，客户版不包含内部成本和利润
- 全量本地 JSON 备份与恢复
- PWA 安装与离线使用

## 本地运行

需要 Node.js 22.13 或更高版本。

```bash
npm install
npm run dev
```

常用校验命令：

```bash
npm test
npm run lint
npm run build
```

## 数据与导出边界

- IndexedDB 是当前业务数据的唯一持久化位置。
- 新建项目会保存创建时的价格库快照；后续修改价格库不会回写历史报价。
- 客户版仅导出 `customerVisible` 的费用项，不导出内部成本、成本调整原因、毛利或最低安全价。
- 内部版含成本和利润信息，只应在公司受控环境中流转。
- 本地备份文件是未加密 JSON，包含敏感价格和成本数据，请存放在受控位置。

## 部署说明

项目使用 vinext 构建，可继续按现有 Cloudflare Workers/Pages 流程部署。本轮开发未执行任何部署操作。
